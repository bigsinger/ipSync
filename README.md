# ipSync
同步设备IP



提示词：

```
使用C#编写一个定时轮询的程序，定时同步本机的ip，功能如下：
1、配置文件是json格式，程序启动后先读取配置文件。
2、启动时先读取一次lastIP(在某个文件中)，缓存到内存变量：lastIP
3、启动后即调用一次函数功能，此后函数功能以子线程的方式定时调用，时间间隔是在配置文件设置的，单位是秒，默认是60秒。
4、函数功能：获取本机的ip地址，如果有ipv6则优先获取ipv6地址，若无则获取ipv4地址，然后与lastIP进行比较，如果发送变化，则调用同步函数。
5、同步函数分别调用两个封装的函数：
	第一个函数是通过邮件的方式发送通知，格式为：[ip]:name=ip
	第二个函数是通过调用前面的ip.php的方式把ip同步出去。
```
